## New Bing Chat Feature Set to Launch on SwiftKey Mobile Typing App

###### [Read update](https://on-screen-recording.techidaily.com/updated-2024-approved-vrecorder-basics-seamless-installation/) 

* Bing Chat has now been [officially announced](https://blogs.bing.com/search/april-2023/Easily-access-the-new-AI-powered-Bing-across-your-favorite-mobile-apps) for SwiftKey on both iPhone and Android devices. The update allows you to ask bring up the usual chat interface, or use Bing to customize your text for different tones.

 Microsoft and Google are eager to integrate their new AI chatbots into as many applications and services as possible. Microsoft's Bing Chat AI is already integrated in apps like [Edge](https://tech-revival.techidaily.com/eliminating-streaming-glitches-on-chatgpt-platform/) and [Skype](https://facebook-clips.techidaily.com/updated-in-2024-maximize-video-impact-with-full-screen-settings/), and now it's coming to SwiftKey.

###  UPDATE: 4/14/23

 Bing Chat has now been [officially announced](https://blogs.bing.com/search/april-2023/Easily-access-the-new-AI-powered-Bing-across-your-favorite-mobile-apps) for SwiftKey on both iPhone and Android devices. The update allows you to ask bring up the usual chat interface, or use Bing to customize your text for different tones.

 SwiftKey is the virtual keyboard for iOS and Android developed by Microsoft, which [it acquired in 2016](https://www.pcworld.com/article/419466/microsoft-confirms-swiftkey-buy-says-it-will-continue-developing-android-and-ios-apps.html). It already has integrations with services like Microsoft To Do, so it's not a surprise that Bing Chat is making an appearance. In the latest [beta releases for the Android app](https://www.anrdoezrs.net/links/3607085/type/dlg/sid/UUhtgUeUpU202153/https://play.google.com/store/apps/details?id=com.touchtype.swiftkey.beta), a Bing button is available in the keyboard's toolbar, which gives you a text box for entering a prompt. The view also expands into a full-size popup with the usual Bing Chat interface.

 Microsoft CTO Pedram Rezaei confirmed on Twitter that that the AI assistant is "slowly rolling out," though there's no indication for when the iOS version will receive the same feature. On Apple devices, the quickest way to access Bing Chat is probably still with the Bing app.

 Microsoft [announced in September 2022](https://android-location-track.techidaily.com/top-9-motorola-moto-g34-5g-monitoring-apps-for-parental-controls-drfone-by-drfone-virtual-android/) that it was discontinuing SwiftKey for iPhone and iPad, but the decision was [later reversed in November](https://youtube-help.techidaily.com/in-2024-icy-images-cozier-cinematics-best-bgs-selection/) due to "popular demand." The Android version has been under active development the whole time, while some of SwiftKey's technology still powers the virtual keyboard on Windows 10 and 11.
