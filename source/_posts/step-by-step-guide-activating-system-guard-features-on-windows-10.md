## Step-by-Step Guide: Activating System Guard Features on Windows 10

System Protection is disabled by default in Windows 10\. If it is disabled, Windows is not able to create restore points which allows you to use System Restore to restore the system. So it is necessary to turn on System Protection.  
  
Follow these steps to turn on System Protection:  
  
1\. Open [Control Panel](https://tools.techidaily.com/drivereasy/download/) .   
  
2\. View by **Small icons** and click on **System** .   
  
![](https://images.drivereasy.com/wp-content/uploads/2017/01/img_5874ae59063e3.jpg)   
  
 3\. Click**System Protection** in left pane.  

![](https://images.drivereasy.com/wp-content/uploads/2017/01/img_5874af17efd1d.jpg) 

  
4\. Under “Protection Settings”  , select the drive that you have Windows installed. (If you didn’t save it to another drive, Windows is installed on C:\\ drive by default.) then click**Configure** button.  
  
![](https://images.drivereasy.com/wp-content/uploads/2017/01/img_587855c91605c.jpg)   
  
 5\. Select option**Turn on system protection** and click**OK** button.  

![](https://images.drivereasy.com/wp-content/uploads/2017/01/img_58785621ed96e.jpg)   
  
 After System Protection is turned on, Windows 10 will create restores points automatiicdally. If you want, you can also[create restore points manually](https://tools.techidaily.com/drivereasy/download/) .
